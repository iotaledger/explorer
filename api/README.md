# Tangle Explorer API

This package facilitates all the services required by the `client` web UIs.

It is a set of services running on NodeJS.

## Pre-requisites

```shell
npm install
```

## Development

This will run the api at <http://localhost:4000>

```shell
npm run start-dev
```
